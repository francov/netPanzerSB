About

About NetPanzerSB

NetPanzerSB is a Server-Browser application for NetPanzer the game. It was developed in collaboration with the maintainers of the game and with the owner of the official game site http://www.netpanzer.info, which provides data for this application via JSON.
The same authors developed a Telegram® Bot which alerts when players are online by sending a message to the public Telegram® channel @Netpanzer Gamers

About NetPanzer

NetPanzer is an isometric view real-time tactical game designed for play across the Internet and over LAN systems.
Players can fight until their units are destroyed, then respawn and keep on going. The game is real-time, but it's based on quick tactical action and unit management. Battles progress quickly and constantly.

Links

Official NetPanzer website: http://www.netpanzer.info. Data source for this application is taken from there.

Telegram® public channel: @Netpanzer Gamers. You can join there to receive alerts when players are online.

NetPanzerSB is released under a GPLv3 license, and source code is available on https://github.com/francov/netPanzerSB.
